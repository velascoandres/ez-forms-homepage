To make use of text entries in the form, the field must first be defined through an object of type InputTextFieldInterface.

InputTextFieldInterface

The InputTextFieldInterface extends of PrincipalFormFieldInterface

Property Description
controlName Field name
label Label for input
placeholder Short hint that descripbes the expected value of an input field
type Object of InputTextTypeInterface:
validators Array of angular validators or EzForms custom validators.

InputTextTypeInterface

The InputTextTypeInterface extends of PrincipalTypeInterface.

Property Description
typeName Type name could be "input-text" for simple input of "textArea" for textarea
maxLength the input value max length
minLenght the input value min length
class Is used to hide the characters entered. The only value for this property is "password".
